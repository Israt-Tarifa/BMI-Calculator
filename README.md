# 📱 BMI Calculator (Flutter)

A sleek, minimal, and user-friendly **Body Mass Index (BMI)** calculator built with the **Flutter** framework. This project demonstrates clean UI implementation and efficient state handling in Dart.

## ✨ Features
* **Precision Input:** Seamlessly calculate BMI using height (m) and weight (kg).
* **Instant Feedback:** Real-time calculation using optimized logic.
* **Minimalist UI:** Designed with **Material 3** principles for a modern, clean look.
* **Lightweight & Fast:** Zero external library dependencies for maximum performance.

## 🛠️ Tech Stack
* **Framework:** Flutter
* **Language:** Dart
* **Design:** Material Design 3

## 📊 Technical Logic
The application performs calculations based on the standard BMI formula:

$$BMI = \frac{weight (kg)}{height (m)^2}$$
